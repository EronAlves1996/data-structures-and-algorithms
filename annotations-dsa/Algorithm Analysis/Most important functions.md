
## Constant

`f(n) = c`
`g(n) = 1`

## Logarithm Function

`f(n) = logn`

## Linear Function

`f(n) = n`

##  N-log-n

`f(n) = nlogn`

## Quadratic Function

`f(n) = n^2`

## Cubic Function

`f(n) = n^3`

## Exponential Function

`f(n) = b^n`

We would like data structures operations to run in times proportional to the constant or logarithm function and algorithms running in linear or n-log-n time.
